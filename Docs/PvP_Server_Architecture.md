# [스카이 윙(Sky Wings)] 3대3 PVP 매칭 및 동기화 서버 아키텍처 구성안

## 1. 서버 시스템 아키텍처 개요
'스카이 윙'은 물리 엔진(PhysX) 기반의 정밀한 기동과 슈팅이 결합된 게임입니다. 멀티플레이어 환경에서 클라이언트 간의 물리적 결과가 어긋나지 않도록 **권위 있는 서버(Authoritative Server)** 기반의 Client-Server 아키텍처를 채택합니다.

- **Matchmaking Server**: 유저의 MMR(Match Making Rating)과 계급 데이터를 기반으로 3대3 플레이어를 그룹화합니다.
- **Dedicated Game Server (DGS)**: 물리 엔진이 포함된 유니티 헤드리스(Headless) 서버 스탠스를 띄워 실제 게임의 물리 계산 및 피격 판정을 총괄합니다.

---

## 2. 3대3 PVP 매칭 로직 구성
1. **유저 요청**: 유저가 PVP 모드 진입 시 매치메이킹 큐에 등록.
2. **풀(Pool) 관리**: Redis 등을 활용하여 큐에 들어온 유저의 '계급 체계(훈련병~정규군)' 및 '기체 성능 점수(Gear Score)'를 기반으로 점수대가 유사한 6명을 탐색.
3. **세션 할당**: 6명이 모이면 DGS(Dedicated Game Server) 클러스터에서 빈 인스턴스를 하나 할당받고, 6명의 클라이언트에게 해당 DGS의 IP 및 Port 번호를 응답.
4. **로딩 동기화**: 모든 클라이언트가 접속을 완료하고 맵과 스크립터블 오브젝트(무장, 스탯) 데이터를 로드할 때까지 대기한 후 게임 시작.

---

## 3. 클라이언트-서버 간 데이터 동기화 방식 (State Synchronization)

비행 슈팅 게임 특성상 이동 속도가 매우 빠르고 3차원 공간을 사용하므로 핑(Ping)에 의한 지연을 최소화하는 하이브리드 동기화 기법을 사용합니다.

### A. 클라이언트 예측 (Client Prediction)
- 클라이언트에서는 유저가 입력한 조작(이동, 롤링 등)을 즉시 로컬 `PlayerController`에 적용하여 화면에 렌더링합니다(물리 반응 즉각성 확보).
- 이와 동시에 해당 입력(Input Data: 방향키, 부스트, 롤링 등 시간 스탬프 포함)을 서버로 전송합니다.

### B. 서버 검증 및 물리 연산 (Server Authoritative)
- 서버는 클라이언트에서 전달받은 조작 입력을 가상의 PhysX 씬에서 계산합니다. 
- 기체의 무게, 부품 장착 위치에 따른 관성 및 질량 중심(Center of Mass) 변화를 서버가 직접 계산하여 부정행위(Speed 핵, 무적 핵 등)를 방지합니다.

### C. 서버 리콘실리에이션 (Server Reconciliation)
- 서버는 계산된 공식 위치 및 상태(State Snapshot: 위치, 회전, 속도, 각속도)를 초당 20~30회(Tickrate 20~30Hz) 클라이언트들에게 브로드캐스팅합니다.
- 클라이언트는 서버의 응답을 받으면 로컬에서 예측했던 기체의 위치와 서버 위치 간의 오차를 계산합니다. 오차가 허용 범위 이내라면 부드럽게 보간(Interpolation)하고, 허용 범위를 넘어서면 서버의 위치로 강제 스냅(Snap) 시킵니다.

### D. 실시간 데드 레커닝 (Dead Reckoning) & 추측항법
- 다른 플레이어(적군 및 아군 5명)의 위치는 클라이언트가 직접 조작할 수 없으므로, 가장 마지막에 서버로부터 받은 속도와 가속도, 각속도 벡터를 바탕으로 다음 위치를 추측하여 부드럽게 렌더링(Entity Interpolation)합니다. 

## 4. 무장 및 피격 판정 (Hit Registration)
- **레이캐스트 및 투사체**: 유도탄, 레이저 등의 무기는 클라이언트에서 발사 이펙트를 즉시 보여주지만, 데미지 처리는 서버에서만 적용됩니다(Server-Side Hit Detection). 
- 높은 핑을 가진 유저를 위해 서버는 과거의 틱(Tick) 상태를 버퍼에 저장하고, 발사한 순간의 지연(Ping)을 역추적하여 피격 여부를 검증하는 **Lag Compensation(지연 보상)** 기법을 활용합니다.
